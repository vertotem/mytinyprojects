# 土地革命战争时期(1927.8—1937.7)

>
> ![](./003.jpg)
>
> 土地革命战争又称“第二次国内革命战争”，是中国共产党领导人民反对蒋介石反动统治的革命战争，从1927年蒋介石、汪精卫发动反革命政变开始，到1937年抗日战争爆发后结束。
