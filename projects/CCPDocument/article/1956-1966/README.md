# 开始全面建设社会主义时期(1956.9—1966.4)

>
> ![](../../pic/007.jpg)
>
> 1956年9月15日至27日，中国共产党第八次全国代表大会在北京举行。这次大会是在我国社会主义改造基本完成，党面临着新的形势和任务的情况下召开的。八大正确地分析了国内形势和国内主要矛盾的变化，提出了党在今后的根本任务，确定了以经济建设为中心的建设社会主义现代化国家的基本指导思想和路线。
