# 抗日战争时期(1937.7—1945.8)

>
> ![](../../pic/004.jpg)
>
> 二十世纪三十年代，第一次世界大战后走上法西斯主义道路的德、意、日三国企图重新瓜分世界，先后结为反共同盟，成为欧洲和亚洲的战争策源地。经过长期的准备，日本帝国主义于1937年7月公然发动大规模的全面侵华战争。7月7日夜，日本侵略军在北平西南的卢沟桥附近，突然向中国驻军进攻，中国官兵奋起抵抗。中华民族全面抗战从此开始。
